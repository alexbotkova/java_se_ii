# JAVA SE II
CLI tool for arrays operations.

## Requirements
- Java 24[pom.xml](pom.xml)
- Apache Maven

## Build the Project

```bash
mvn clean install
``` 

## Run the GUI Application

```bash
mvn exec:java
``` 

## Run Unit Tests

```bash
mvn test
``` 

## Generate Javadoc

```bash
mvn clean javadoc:javadoc
```
